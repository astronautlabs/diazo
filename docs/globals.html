Const MULTIPLE_VALUES

MULTIPLE_VALUES: unique symbol = Symbol('Multiple values')
  • Defined in lib/accessor.ts:11

This symbol is returned by Accessor when the set of objects being queried disagree about the value of a the queried property path. This is used by the Diazo editor's Properties sidebar to know when to show "Multiple values" within a given property view.

template

  • template(context: any, content: string): string
    • Defined in lib/template.ts:14

    Given a context object and a template string, interpolates any present template directives and outputs the resulting string. Template directives can take one of the following formats:

    • {{path.to.property}} -- takes the value from the given jsonpath and uses it to replace the template directive
    • {{path.to.property | default}} -- same as the above, but if path.to.property is falsey, uses the text "default" instead.

    Parameters

    • context: any
    • content: string

    Returns string
