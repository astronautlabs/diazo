@astronautlabs/firegraph

@astronautlabs/firegraph

An MIT-licensed slot-driven directed-acyclical-graph (DAG) editor for Angular. This component was built for use in Astronaut Labs' Livefire product suite because there was no good existing open-source editor. We wanted to fix that, so we hope you find lots of great ways to use this editor in your projects!

Firegraph is inspired by Blueprint, the visual programming language of Unreal Engine. Firegraph itself is just the editor that would power such an experience, and the library itself is completely domain-agnostic. You can use Firegraph in your apps to introduce visual node-graph functionality in any domain where it makes sense to apply it, and we encourage you to do so!
